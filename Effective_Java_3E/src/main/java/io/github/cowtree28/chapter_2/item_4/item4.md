# 4. 인스턴스화를 막으려거든 private 생성자를 사용하라
> 정적 메서드, 필드만 담으려는 클래스는 private 생성자로 기본 생성자 막기

## 인스턴스화 막기
인스턴스화를 막아야 하는 경우가 있다(웬만하면 하지 말자). 예를 들어  
`java.lang.Math`와 같은 값, 팩토리 메서드를 모아놓은 클래스라던가
`java.Collections`처럼 인터페이스를 구현하는 객체를 생성해주는 메서드를 모아놓은 클래스도 있을수 있다.  
### 추상클래스로 만들어 인스턴스 막기?
겉으로 보기에 막을수 있어보이지만, 상속받아 우회하면 그만이다.  
그럼 어떻게 해결할까?
### private 생성자 쓰기
간단히 private 생성자로 기본 생성자가 만들어지는걸 막아 구현이 가능하다.
